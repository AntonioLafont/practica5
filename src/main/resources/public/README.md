# Vinylcrate

Tienda de vinilos online. Frontend responsive con HTML5, CSS3 y diseño temático Spotify.

## Contenido

4 páginas interconectadas:

- **Inicio** - Página principal con presentación de la tienda
- **Productos** - Catálogo de 12 álbumes disponibles para comprar
- **Carrito** - Carrito de compra con formulario de envío y métodos de pago
- **Contacto** - Información de la tienda, formulario de contacto y ubicación en mapa

## Elementos Incluidos

Cada página contiene:
- Encabezado con navegación
- Elementos HTML: títulos, párrafos, tablas, formularios
- Estilos CSS responsivos (mobile-first)
- Multimedia: imágenes, Google Maps embebido
- Pie de página

## Diseño

- Tema Spotify: verde primario (#1DB954)
- Responsivo y adaptable a móviles
- Navegación consistente entre páginas
- Párrafos (`<p>`)
- Tablas (`<table>`)
- Formularios (`<form>`)
- Listas (`<ul>`, `<ol>`)
- Enlaces (`<a>`)
- Multimedia (iconos, espacios para imágenes)

✅ **CSS Uniforme y Personal**
- Estilo consistente en todas las páginas
- Colores y tipografía elegidos
- Responsive design
- Animaciones y efectos

✅ **Código Limpio**
- HTML semántico
- CSS organizado
- JavaScript modular
- Comentarios explicativos

## Navegador Soportados
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Notas
- La aplicación requiere que Spring Boot esté ejecutándose
- Los datos se procesan a través de la API REST del backend
- El localStorage solo almacena referencias locales
- La compatibilidad es total en navegadores modernos
